# Java Web Applications Collection

A comprehensive collection of Java-based web applications demonstrating core concepts of backend development, including User Authentication, the MVC Design Pattern, and database connectivity.

## 🚀 Projects Included

### 1. MVC Design Pattern Project
This project demonstrates the *Model-View-Controller (MVC)* architectural pattern to separate business logic, user interface, and data control.
* *Technologies:* Java, Servlets, JSP.
* *Key Feature:* Clean separation of concerns for better maintainability.

### 2. Login & Registration System
A full-stack implementation of a user management system.
* *Features:* User registration, secure login validation, and profile management.
* *Backend:* Java Servlets & JSP.
* *Database:* MySQL with JDBC for persistent data storage.

### 3. Basic Login Page
A lightweight implementation focusing on the fundamentals of handling HTTP POST requests and session management.

---

## 🛠️ Tech Stack
* *Language:* Java (JDK 8+)
* *Web Framework:* Java Servlets & JavaServer Pages (JSP)
* *Server:* Apache Tomcat v10.1
* *Architecture:* MVC (Model-View-Controller)
* *Frontend:* HTML, CSS
* *Database:* MySQL
